## 清除PWA缓存

添加了两个缓存方案

1、service-worker.js文件内更换cacheName版本号会自动清除上个版本设置的缓存文件

2、主页index.html文件内，找到注释<!-- 添加版本号-每次更新要改版本号才会刷新缓存 -->更新下面自己设置缓存的文件的版本号

3、清除本地浏览器缓存
